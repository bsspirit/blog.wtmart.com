**GIT常用命令**

*   git init  初始化项目
*   git clone  复制项目
*   git add  untrack到unstage
*   git commit  unstage到stage
*   git checkout   切换分支
*   git branch  查看分支
*   git status  查看状态
*   git log  查看日志
*   git merge  汇总合并提交
*   git rebase  依次合并提交
*   git diff  比较不同点
*   git gui  打开GUI界面
*   git reset  设置分支的head
*   git push  提交到远程
*   git pull  从远程下载
*   git remote add  配置远程库的信息
